📋 Étapes pour déployer la Web App :

  1. Dans Apps Script, ajoutez un nouveau fichier WebApp.gs
  2. Collez le contenu du fichier WebApp.gs fourni
  3. Cliquez Déployer → Nouveau déploiement
  4. Type : Application Web
  5. Exécuter en tant que : Moi
  6. Accès : Tout le monde
  7. Copiez l'URL et collez-la ci-dessus
